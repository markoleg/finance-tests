# Тести ЄФВВ · Облік та фінанси

Безкоштовні пробні тести для підготовки до **Єдиного фахового вступного випробування (ЄФВВ)**
зі спеціальності «Облік та фінанси» (вступ до магістратури), а також **формульний лист**
за розділами 1–2 програми.

Сайт повністю статичний (HTML + CSS + JS, без бекенду) і розрахований на GitHub Pages.

## 🌐 Живий сайт

> Після ввімкнення GitHub Pages: `https://<користувач>.github.io/<репозиторій>/`

## Що всередині

- **7 пробних варіантів 2026** різної складності, по 140 завдань, з миттєвою перевіркою
  та поясненнями:
  - Варіанти 1–2 — легкий рівень
  - Варіанти 3–4 — середній рівень
  - Варіант 5 — складний рівень
  - Варіанти 6–7 — екзаменаційний рівень (за зразком офіційного тесту 2024)
- **Формульний лист** — фінансова звітність (НП(С)БО) та фінансовий аналіз, з пошуком.

Завдання укладені за офіційною програмою предметного тесту (наказ МОН від 15.10.2025 № 1362).
Це матеріали **для самопідготовки**, вони не є офіційними завданнями ЄФВВ.

## Структура

```
build.py                 # збирає сайт у docs/ з банків питань
generators/              # генератори банків питань
  build_questions.py     #   єдина точка входу — відтворює всі questions2025*.json
  gen2025*.py, enrich.py #   генератори варіантів + додавання пояснень
questions2025*.json      # банки питань (вихідні дані для build.py)
docs/                    # ← корінь сайту для GitHub Pages
  index.html             #   головна сторінка
  variant-1..7.html      #   тести
  formulas.html          #   формульний лист
  assets/                #   спільні стилі (quiz.css), рушій (quiz.js), favicon
```

## Як перезібрати локально

```bash
python generators/build_questions.py   # (за потреби) відтворити банки питань
python build.py                         # зібрати сайт у docs/
```

Для локального перегляду:

```bash
python -m http.server -d docs 8000
# відкрити http://localhost:8000/
```

## Публікація на GitHub Pages

1. Створити репозиторій на GitHub і запушити цей код.
2. Settings → Pages → Build and deployment → **Deploy from a branch**.
3. Branch: `main`, папка: **`/docs`** → Save.
4. За хвилину сайт буде доступний за адресою з розділу Pages.
